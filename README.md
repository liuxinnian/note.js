note.js
============

基于node.js + mongodb 的web笔记本，基于 express3.0 mvc架构，twitter-bootstrap 前端框架。可以做为完整学习范例。

初期开发阶段，不建议下载投入使用。

演示地址: http://notejs-5293.onmodulus.net/    (runing on [modulus.io](http://modulus.io/))
